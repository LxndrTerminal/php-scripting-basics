# PHP Scripting Basics

## Project Description
This project is designed to provide a comprehensive introduction to PHP scripting. It covers the fundamental concepts, syntax, and functions needed to get started with PHP programming. Whether you are a beginner or looking to brush up on your PHP skills, this repository serves as a great resource.

## Usage
To use the scripts in this repository, clone the repository to your local machine and make sure you have PHP installed.

1. Clone the repository:
   ```bash
   git clone https://github.com/LxndrTerminal/php-scripting-basics.git
2. Navigate to scripts folder
3. Run in terminal using
   ```code
   php filename.php
